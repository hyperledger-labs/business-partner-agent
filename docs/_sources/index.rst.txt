======================
Business Partner Agent
======================

This documentation is designed to help you onboard with the Business Partner Agent.  It covers the Business Partner Agent as a concept, explains the tech stack, and guides you through all the features it has to offer.

As with any open-source endeavor, feel free to suggest new topics and provide examples wherever you think it might be helpful.  All documentation is editable via GitHub -- if you're unsure how, `follow these instructions <https://github.com/hyperledger-labs/business-partner-agent/blob/main/docs/README.md>`_.
